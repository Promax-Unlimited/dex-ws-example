# Dex WebSocket client (TypeScript)

> **Note:** This is a public repository providing a minimal reference implementation for connecting to the Dex WebSocket API. It serves as a starting point for developers integrating with the Dex platform via WebSocket connections.

## Quick start

1) Install deps (requires network): `npm install`
2) Set your token: `export DEX_TOKEN=your_token_here`
3) Set your token: `export DEX_BASE_URL=dex_base_url`
4) Run the example: `npm run dev`
